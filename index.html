Git e GitHub

Git

O Git é um sistema de controle de versão que permite que desenvolvedores acompanhem alterações no código, trabalhem de forma colaborativa e mantenham um histórico completo das modificações realizadas em um projeto. Com ele, é possível criar diferentes versões de um mesmo projeto, testar alterações sem comprometer o código principal e reverter mudanças quando necessário.

GitHub

O GitHub é uma plataforma online utilizada para hospedar e compartilhar projetos de software. Funcionando com base no sistema de controle de versão Git, permite que desenvolvedores hospedem seus projetos para estudo ou até mesmo como portfólio profissional.
